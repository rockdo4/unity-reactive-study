# 유니티 리액티브 프로그래밍 (R3) 수업 개요

리액티브 프로그래밍의 **일반 이론**과 **R3 사용법**을 다루는 독립 강의입니다. 디자인 패턴 강의의 부속이 아니라, 옵저버 패턴 기초부터 R3 실전 활용까지 자족적으로 다룹니다. 모든 차시는 **이론 → 따라하기 → (실습)** 순서로 진행합니다.

> 기준 버전: **R3** (`com.cysharp.r3`, 수업 시점 최신 버전) · Unity 6.3 / URP 17
> 공식 저장소: <https://github.com/Cysharp/R3>

## 수업 운영 방식

- 각 차시는 **이론 설명 → 강사 시연·학생 따라하기 → 실습** 순서를 따릅니다.
- 1부 데모는 한 씬(`ReactiveBasics`) 안에 차시별 패널(1~5)로 구성되어, 그날 배운 패널만 보면 됩니다.
- 2부는 학생이 교안의 단계(1~5단계)를 따라 **직접 코드를 작성**하는 따라하기 수업입니다. 완성본 씬은 막혔을 때 비교·확인용입니다.

## 데모 프로젝트 준비

1. 이 저장소를 클론하고 Unity 6.3으로 엽니다. R3는 **이미 설치되어 있습니다** (NuGet `Assets/Packages/` + Package Manager `R3.Unity`).
2. 데모 씬은 에디터 메뉴로 자동 생성합니다.
    - 1부: `Tools > Reactive Study > 1부 기초 데모 씬 생성` → `Assets/Scenes/ReactiveBasics.unity`
    - 2부: `Tools > Reactive Study > 2부 클리커 씬 생성` → `Assets/Scenes/ReactiveClicker.unity` (완성본)
3. 데모 코드 위치: `Assets/Scripts/ReactiveBasics/`(1부), `Assets/Scripts/ClickerGame/`(2부).

> ⚠️ 새 프로젝트에 R3를 설치하는 절차(NuGetForUnity + git URL)는 [교안 02](교안/02.%20R3%20시작하기%20—%20Observable%20만들기.md)의 1장에서 단계별로 다룹니다.

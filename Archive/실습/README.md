# 보관: 실습(연습문제) 2종

R3 강의는 **이론 → 따라하기** 흐름으로 운영하며, 자력 연습문제(실습)는 강의 구성에서 **생략**했습니다. 기존 실습 문서 2종을 이 폴더에 보관합니다(복원 가능). 이 폴더는 저장소 루트에 있어 Unity가 임포트하지 않습니다.

## 내용물

| 보관 파일                              | 원래 위치    | 대응                        |
| -------------------------------------- | ------------ | --------------------------- |
| `01. 1부 실습 — 연산자와 수명 관리.md` | `Docs/실습/` | 1부(교안 02~05) 마무리 실습 |
| `02. 2부 실습 — 클리커 확장.md`        | `Docs/실습/` | 2부(교안 06) 마무리 실습    |

## 복원 방법

1. 위 파일들을 `Docs/실습/`으로 다시 옮깁니다.
2. 실습으로 되돌릴 곳의 링크·문장을 복구합니다(이 폴더를 만든 커밋의 diff 역참조: `git log --follow -- "Archive/실습"`). 주로 `Docs/00. 수업 개요.md`(커리큘럼 표·차시 흐름), `Docs/강사용/00. 강사 사전 학습 가이드.md` 또는 운영 안내, `README.md`.
3. 실습은 학생 자력 과제이므로 데모 패키지 배포 대상이 아닙니다(교안·문제 본문만 노션 등으로 공유).
